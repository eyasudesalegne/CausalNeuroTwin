# OpenNeuro ds004024, version 1.0.1

## Registration status

CausalNeuroTwin pins this dataset to DOI `10.18112/openneuro.ds004024.v1.0.1` and BIDS version `1.6.0`. The source metadata declare a raw CC0 dataset with TMS-EEG, structural MRI, resting-state fMRI for a subset, and HARDI diffusion MRI for a subset.

Phase 03 validates dataset identity and local directory layout. It does **not** claim that NIfTI, EEG, DWI, or fMRI payloads are readable, complete, BIDS-valid, or scientifically usable. Those checks belong to Phase 04.

## Local placement

The default non-sensitive layout is:

```text
${CAUSALNEUROTWIN_DATA_ROOT}/openneuro/ds004024-v1.0.1/
```

Raw data must remain outside Git.

## Acquisition options

Use an official OpenNeuro download method or the versioned DataLad mirror. Pin the acquired snapshot to version `1.0.1`; do not validate an unpinned moving branch as though it were the registered version.

The repository deliberately does not automate a multi-gigabyte download in CI.

## Validate the local registration

```bash
causalneurotwin dataset validate \
  --registry configs/data/openneuro_ds004024.yaml \
  --dataset-root /path/to/ds004024-v1.0.1 \
  --output-dir runs/dataset-registration-ds004024
```

Or set `CAUSALNEUROTWIN_DATA_ROOT` and omit `--dataset-root`.

## Expected identity

- dataset ID: `ds004024`
- version: `1.0.1`
- licence: `CC0`
- BIDS version: `1.6.0`
- dataset type: `raw`
- expected participant count: `13`
- all participants: at least one T1-weighted anatomical filename
- optional per subject: resting-state fMRI and diffusion MRI

## Ethics and responsible use

The source metadata report ethics approval `STU00204239`. Public availability does not remove the obligation to minimise participant-level exposure, avoid re-identification, honour acknowledgement requirements, and limit claims to research use. CausalNeuroTwin does not commit raw data, participant attributes, or locally derived subject-level reports.
